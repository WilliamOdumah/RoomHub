(route) Create an expense

Route:
MethodPath
POST transaction/create-expense
Body Parameters:
Name Type Description
name String The transaction name
price int The cost of the expense
payer String The user that created the transaction
contributors list The list of people that contributed to the transaction
date String Date representing the date the transaction was created. (yyyy-mm-dd)
Response:
Name Type Description
message String See description of the different status codes
Response Code:
Type Description
200 Expense created successfully
404 One or more contributors no longer belongs to this room
404 User does not exist
422 Invalid Transaction Name
422 Invalid Payer Name
422 Invalid Price- Price must be a positive number
422 Invalid Date
422 Invalid Contributors - Contributors must be a non empty list
500 Backend error from the database

MethodPath
POST transaction/settle-up
Body Parameters:
Name Type Description

Response:
Name Type Description
message String See description of the different status codes

Type Description

422 Invalid Settle Up amount- Settle Up amount must be a positive number
